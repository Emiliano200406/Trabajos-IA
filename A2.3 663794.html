A2.3 Modelos de ensamble, SVM y redes neuronales¶

1 Descripcion del problema¶

En esta actividad se analiza un problema de clasificación supervisada utilizando el conjunto de datos Adult Income disponible en el UCI Machine Learning Repository. Este dataset contiene información demográfica y laboral de individuos recopilada a partir del censo de Estados Unidos.

El objetivo del análisis consiste en predecir si el ingreso anual de una persona supera los 50,000 dólares, utilizando variables socioeconómicas como edad, nivel educativo, tipo de trabajo, estado civil y número de horas trabajadas por semana.

Este problema resulta adecuado para evaluar distintos algoritmos de aprendizaje automático, ya que combina variables numéricas y categóricas y cuenta con un tamaño de muestra suficientemente grande para entrenar modelos más complejos. En particular, se compararán modelos de Random Forest, Boosting, Support Vector Machine y redes neuronales, con el propósito de analizar cuál ofrece el mejor desempeño predictivo.

1.1 Variable objetivo¶

La variable objetivo del modelo es income, que indica si el ingreso anual de la persona es:

  • 50K (ingreso alto)

  • ≤50K (ingreso bajo)

Este tipo de variable binaria permite formular el problema como una tarea de clasificación supervisada.

2 Limpieza y preparación de datos¶

Antes de entrenar los modelos de aprendizaje automático, es necesario realizar un proceso de limpieza y preparación de los datos. Este proceso incluye la identificación de valores faltantes, la eliminación o tratamiento de registros incompletos y la transformación de variables categóricas a un formato numérico que pueda ser interpretado por los algoritmos de aprendizaje automático.

En el conjunto de datos Adult Income, algunos valores faltantes están representados por el símbolo "?". Estos registros deben ser tratados adecuadamente para evitar que afecten el desempeño de los modelos.

Posteriormente, se separa la variable objetivo (income) del resto de las variables predictoras. Finalmente, el conjunto de datos se divide en dos subconjuntos: uno para entrenamiento y otro para prueba, lo que permite evaluar el desempeño de los modelos sobre datos que no fueron utilizados durante el entrenamiento.

3. Modelo Random Forest¶

El primer modelo implementado para resolver el problema de clasificación fue Random Forest, un algoritmo de aprendizaje automático basado en conjuntos (ensemble learning). Este método construye múltiples árboles de decisión durante el entrenamiento y combina sus predicciones para obtener un resultado final más robusto.

Random Forest presenta varias ventajas, entre ellas:

  • Reduce el riesgo de overfitting presente en árboles individuales.
  • Maneja adecuadamente grandes cantidades de variables.
  • Puede capturar relaciones no lineales entre las variables.

En este estudio, el modelo fue entrenado utilizando el conjunto de datos de entrenamiento y posteriormente evaluado sobre el conjunto de prueba para medir su desempeño predictivo.

In [8]:
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import accuracy_score

rf = RandomForestClassifier(
    n_estimators=200,
    random_state=42
)

rf.fit(X_train, y_train)

y_pred_rf = rf.predict(X_test)

acc_rf = accuracy_score(y_test, y_pred_rf)

print("Accuracy Random Forest:", acc_rf)

Accuracy Random Forest: 0.8528014145209415

3.1 Evaluación del modelo Random Forest¶

Para evaluar el desempeño del modelo Random Forest se utilizó el conjunto de prueba, el cual contiene observaciones que no fueron utilizadas durante el entrenamiento del modelo. Esto permite medir la capacidad del algoritmo para generalizar a nuevos datos.

El modelo obtuvo una precisión (accuracy) de 0.8528, lo que indica que aproximadamente el 85.28% de las observaciones fueron clasificadas correctamente.

Este resultado sugiere que el modelo es capaz de identificar patrones relevantes entre las variables sociodemográficas y el nivel de ingreso de los individuos. Sin embargo, para comprender mejor el desempeño del modelo es necesario analizar métricas adicionales como la matriz de confusión, la cual permite observar el número de clasificaciones correctas e incorrectas para cada categoría.

4 Support Vector Machine (SVM)¶

El segundo modelo implementado fue Support Vector Machine (SVM), un algoritmo de aprendizaje supervisado ampliamente utilizado para problemas de clasificación. Este modelo busca encontrar el hiperplano que mejor separa las clases en el espacio de características, maximizando la distancia entre los puntos de datos más cercanos de cada clase, conocidos como vectores de soporte.

Para este análisis se utilizó un kernel radial (RBF), el cual permite capturar relaciones no lineales entre las variables. Antes de entrenar el modelo, se realizó un proceso de escalamiento de las variables, ya que SVM es sensible a las diferencias de escala entre las características.

5. Gradient Boosting¶

El siguiente modelo implementado fue Gradient Boosting, un método de aprendizaje basado en conjuntos (ensemble learning) que construye modelos secuenciales donde cada nuevo modelo intenta corregir los errores cometidos por los anteriores.

A diferencia de Random Forest, donde los árboles se construyen de manera independiente, en Gradient Boosting los árboles se construyen de forma secuencial, lo que permite mejorar progresivamente el desempeño del modelo.

Este enfoque suele ofrecer un alto rendimiento en problemas de clasificación tabular como el presente.

In [18]:

6. Red neuronal¶

El último modelo evaluado fue una red neuronal artificial implementada con Keras. Este tipo de modelo permite aprender relaciones no lineales entre las variables mediante capas de neuronas interconectadas.

Para este análisis se construyó una red neuronal secuencial sencilla, compuesta por una capa de entrada, una capa oculta con función de activación ReLU y una capa de salida con función de activación sigmoide, adecuada para problemas de clasificación binaria.

Debido a que las redes neuronales son sensibles a la escala de los datos, se utilizaron las variables previamente estandarizadas. El modelo fue entrenado con el conjunto de entrenamiento y posteriormente evaluado con el conjunto de prueba.

In [23]:
import tensorflow as tf
from tensorflow import keras
from tensorflow.keras import layers
from sklearn.metrics import accuracy_score

tf.random.set_seed(42)

model = keras.Sequential([
    layers.Dense(32, activation='relu', input_shape=(X_train_scaled.shape[1],)),
    layers.Dense(16, activation='relu'),
    layers.Dense(1, activation='sigmoid')
])

model.compile(
    optimizer='adam',
    loss='binary_crossentropy',
    metrics=['accuracy']
)

history = model.fit(
    X_train_scaled, y_train,
    epochs=20,
    batch_size=32,
    validation_split=0.2,
    verbose=1
)

import numpy as np

y_pred_nn = (y_prob_nn > 0.5).astype(int)

print("Accuracy Red Neuronal (Keras):", acc_nn)

7. Evaluacion y comparacion de desempeño¶

En este estudio se evaluaron cuatro modelos de aprendizaje automático con el objetivo de predecir si el ingreso anual de una persona supera los 50,000 dólares. Los modelos analizados fueron Random Forest, Support Vector Machine (SVM), Gradient Boosting y una red neuronal implementada con Keras. Cada modelo fue entrenado utilizando el conjunto de entrenamiento y posteriormente evaluado con el conjunto de prueba para comparar su desempeño.

La gráfica muestra la comparación del accuracy obtenido por cada modelo. El eje horizontal representa los distintos algoritmos evaluados, mientras que el eje vertical muestra el valor de accuracy obtenido en el conjunto de prueba.

A partir de esta visualización se puede observar que Gradient Boosting obtuvo el mayor nivel de precisión, alcanzando aproximadamente un 86.3% de clasificaciones correctas, lo que lo convierte en el modelo con mejor desempeño entre los evaluados. Esto se debe a que este algoritmo construye los árboles de decisión de manera secuencial, permitiendo que cada nuevo modelo corrija los errores cometidos por los anteriores.

Por otro lado, Random Forest también mostró un desempeño sólido, con un accuracy cercano al 85%. Este modelo es robusto debido a que combina múltiples árboles de decisión entrenados de manera independiente, reduciendo el riesgo de sobreajuste y mejorando la estabilidad de las predicciones.

Los modelos SVM y la red neuronal presentaron resultados ligeramente inferiores, aunque aún competitivos. Ambos algoritmos alcanzaron valores de accuracy cercanos al 84–85%, lo que indica que también logran capturar patrones relevantes dentro del conjunto de datos.

En general, las diferencias observadas entre los modelos son moderadas, lo que sugiere que el problema de clasificación puede ser abordado de manera efectiva por distintos enfoques de aprendizaje automático. Sin embargo, los modelos basados en ensemble learning, particularmente Gradient Boosting, demostraron una ligera ventaja en términos de desempeño predictivo.

8. Análisis crítico y conclusiones¶

Los resultados obtenidos permiten realizar una reflexión sobre el comportamiento de los distintos modelos de aprendizaje automático evaluados. En general, todos los algoritmos lograron un desempeño relativamente alto en la tarea de clasificación, con valores de accuracy superiores al 84%. Esto indica que el conjunto de datos contiene patrones claros que pueden ser identificados por diferentes enfoques de modelado.

Entre los modelos evaluados, Gradient Boosting obtuvo el mejor desempeño, tanto en términos de accuracy como en el área bajo la curva ROC. Esto sugiere que el enfoque secuencial utilizado por este algoritmo, en el que cada nuevo árbol se entrena para corregir los errores de los anteriores, permite capturar de manera más precisa las relaciones presentes en los datos. Sin embargo, la mejora observada respecto a otros modelos no fue extremadamente grande, lo que indica que el problema puede ser abordado de forma efectiva por distintos tipos de algoritmos.

Un aspecto importante a considerar es la relación entre complejidad del modelo y mejora en el desempeño. Aunque Gradient Boosting presentó los mejores resultados, modelos como Random Forest lograron un rendimiento muy cercano utilizando una estructura más sencilla y con menor riesgo de sobreajuste. Esto demuestra que en algunos casos aumentar la complejidad del modelo no necesariamente produce mejoras drásticas en el desempeño.

Otro punto relevante es el riesgo de sobreajuste (overfitting). Los modelos más complejos, especialmente las redes neuronales y los métodos basados en boosting, pueden adaptarse demasiado a los datos de entrenamiento si no se controlan adecuadamente sus parámetros. En este análisis no se observaron indicios claros de sobreajuste, pero en aplicaciones reales sería recomendable utilizar técnicas adicionales de validación para garantizar que el modelo generalice correctamente a nuevos datos.

También es importante considerar la interpretabilidad de los modelos. Algoritmos como Random Forest permiten analizar la importancia de las variables, lo que facilita comprender qué factores influyen en las predicciones del modelo. En contraste, las redes neuronales tienden a ser menos interpretables, ya que su funcionamiento interno es más complejo y difícil de explicar.

Finalmente, la elección del modelo más adecuado depende del contexto de aplicación. Si el objetivo principal es maximizar el desempeño predictivo, Gradient Boosting representa una opción muy sólida. Sin embargo, si se requiere mayor interpretabilidad o menor complejidad computacional, modelos como Random Forest o incluso SVM pueden ser alternativas adecuadas.

En conclusión, este análisis demuestra que diferentes enfoques de aprendizaje automático pueden ofrecer resultados competitivos en problemas de clasificación tabular. La selección final del modelo debe considerar no solo las métricas de desempeño, sino también aspectos como la interpretabilidad, la complejidad del modelo y las necesidades específicas del problema que se desea resolver.
