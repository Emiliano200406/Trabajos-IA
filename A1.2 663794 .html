Regresin lineal simple

Con base en la exploracin grfica previa, se procede a analizar de manera formal la relacin entre el nivel de felicidad y el producto interno bruto (GDP). Para ello, se plantea un modelo de regresin lineal simple en el que la felicidad es tratada como la variable de respuesta, mientras que el GDP se considera la variable explicativa. Este enfoque permite cuantificar la asociacin entre ambas variables y evaluar si el desempeo econmico de un pas est relacionado con su nivel promedio de felicidad.

El modelo propuesto asume una relacin lineal entre las variables y se expresa como: Felicidad=0+1GDP+ donde 0 corresponde al valor esperado de la felicidad cuando el GDP es cero, 1 representa el cambio promedio en el nivel de felicidad ante un incremento unitario en el GDP, y es el trmino de error que recoge la variabilidad no explicada por el modelo.

La estimacin de los parmetros se realiz mediante el mtodo de mnimos cuadrados ordinarios, utilizando herramientas computacionales para ajustar el modelo a los datos observados.

import pandas as pd
import statsmodels.api as sm

# =========================
# 1) Cargar base de datos limpia
# =========================
df = pd.read_csv(
    "Base de datos Indice de Guini, Corrupcion, Felicidad, GDP, Esperanza de vida.csv"
)

# Columnas numricas
cols_num = ["Felicidad", "GDP", "Esperanza_de_vida", "Indice_Corrupcion", "Indice_Gini"]

# Limpiar decimales con coma y convertir a numrico
for c in cols_num:
    df[c] = (
        df[c]
        .astype(str)
        .str.replace(" ", "", regex=False)
        .str.replace(",", ".", regex=False)
    )
    df[c] = pd.to_numeric(df[c], errors="coerce")

# Eliminar filas incompletas
df = df.dropna(subset=cols_num).reset_index(drop=True)

# =========================
# 2) Definir variables
# =========================
y = df["Felicidad"]
X = df[["GDP", "Esperanza_de_vida", "Indice_Corrupcion", "Indice_Gini"]]
X = sm.add_constant(X)

# =========================
# 3) Ajustar modelo mltiple
# =========================
modelo_multiple = sm.OLS(y, X).fit()

# =========================
# 4) Mostrar resultados relevantes
# =========================
print("RESULTADOS RELEVANTES DEL MODELO DE REGRESIN LINEAL MLTIPLE\n")

print(f"Nmero de observaciones: {int(modelo_multiple.nobs)}")
print(f"R: {modelo_multiple.rsquared:.3f}")
print(f"R ajustado: {modelo_multiple.rsquared_adj:.3f}")
print(f"Estadstico F: {modelo_multiple.fvalue:.3f}")
print(f"p-valor (F): {modelo_multiple.f_pvalue:.4f}\n")

print("Coeficientes y significancia individual:")
tabla_coef = pd.DataFrame({
    "Coeficiente": modelo_multiple.params,
    "p-valor": modelo_multiple.pvalues
})
print(tabla_coef)

Interpretacin de los resultados del modelo de regresin lineal mltiple

El modelo de regresin lineal mltiple fue estimado utilizando un total de 141 observaciones, correspondientes a pases con informacin completa en todas las variables consideradas. Al considerar el nmero de variables del modelo, el valor del R2 ajustado disminuye a 0.026, lo cual sugiere que la capacidad explicativa del modelo es limitada.

La prueba global del modelo, evaluada mediante el estadstico F, arroja un valor de 1.933 con un valor p asociado de 0.1084. Dado que este valor p es mayor al nivel de significancia comnmente utilizado (=0.05), no se puede rechazar el modelo nulo. Esto implica que, de manera conjunta, las variables GDP, esperanza de vida, ndice de corrupcin e ndice de Gini no presentan una relacin estadsticamente significativa con el nivel de felicidad en este modelo.

El anlisis de los coeficientes individuales muestra que el coeficiente asociado al GDP es positivo y estadsticamente significativo (p=0.029), lo que sugiere que, manteniendo constantes las dems variables, un mayor GDP se asocia con un mayor nivel de felicidad. En contraste, los coeficientes correspondientes a la esperanza de vida, el ndice de corrupcin y el ndice de Gini no resultan estadsticamente significativos, lo que indica que, dentro de este modelo, su efecto individual sobre la felicidad no puede establecerse con evidencia estadstica suficiente.

En conjunto, estos resultados indican que, aunque el GDP mantiene una asociacin positiva con la felicidad a nivel individual, la inclusin de variables adicionales no mejora de manera significativa la capacidad explicativa del modelo en comparacin con la regresin lineal simple. Esto sugiere que la relacin entre felicidad y factores socioeconmicos es compleja y posiblemente no lineal, o que requiere la incorporacin de variables adicionales o enfoques metodolgicos alternativos para ser explicada de forma ms adecuada.

El modelo de regresin lineal simple permiti identificar una relacin positiva y estadsticamente significativa entre el nivel de felicidad y el producto interno bruto (GDP).
