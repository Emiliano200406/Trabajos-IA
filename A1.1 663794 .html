# Codificar NivelDeObesidad a valores numricos
import pandas as pd
import matplotlib.pyplot as plt

# Cargar datos
data = pd.read_csv("A1.1 Obesidad.csv")

# Filtrar subpoblacin masculina
df = data[data["Sexo"] == "Male"].copy()

# Codificar NivelDeObesidad a valores numricos
categorias = df["NivelDeObesidad"].unique()
mapa = {cat: i for i, cat in enumerate(categorias)}
df["NivelNum"] = df["NivelDeObesidad"].map(mapa)

# Grfica de dispersin
plt.figure(figsize=(10, 5))
plt.scatter(
    x=df["NivelNum"],
    y=df["ComeVegetales"],
    alpha=0.5
)

# Etiquetas y ttulos
plt.xticks(
    ticks=range(len(mapa)),
    labels=mapa.keys(),
    rotation=45
)
plt.xlabel("Nivel de obesidad")
plt.ylabel("Consumo de vegetales")
plt.title("Consumo de vegetales vs Nivel de obesidad (Poblacin masculina)")
plt.show()

El grfico de dispersin obtenido muestra que el consumo de vegetales se distribuye de manera similar entre las distintas categoras de obesidad en la poblacin masculina. No se observa una tendencia clara que indique que un mayor o menor consumo de vegetales est directamente asociado con un nivel especfico de obesidad. Los valores tienden a concentrarse en niveles intermedios de consumo independientemente de la categora de obesidad.

En conclusin, los resultados sugieren que, aunque el consumo de vegetales es un indicador importante de hbitos alimenticios saludables, no resulta suficiente por s solo para explicar las diferencias en los niveles de obesidad dentro de la subpoblacin analizada.
